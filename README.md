# Compteur de mots

Projet Java réalisé dans le cadre du cours Projet Informatique II

## Description

Le programme lit un fichier texte et compte le nombre total de mots qu'il contient. Chaque ligne est lue une par une, séparée par des espaces, et les mots sont additionnés jusqu'à la fin du fichier.

## Structure du projet

- Main.java : point d'entrée du programme, gère la lecture du fichier et le comptage des mots
- text.txt, longText.txt : fichiers texte utilisés pour les tests

## Prérequis

- Java JDK 21
- IntelliJ IDEA

## Lancer le projet

1. Cloner le dépôt
2. Ouvrir le projet dans IntelliJ
3. Placer un fichier texte nommé text.txt (si il n'est pas présent) à la racine du projet
4. Exécuter la classe Main

## Exemple d'exécution

Nombre total de mots : 342
